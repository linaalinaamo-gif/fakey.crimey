# Fakey.Crimey

A simple bio link website that lets you share your online presence from one clean page.

## Features

* 🔗 All your important links in one place
* 🎨 Clean and responsive design
* ⚡ Fast and lightweight
* 📱 Mobile-friendly
* 🌐 Easy to customize

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/linaalinaamo-gif/fakey.crimey.git
```

## Customization

Update the HTML, CSS, and assets to personalize:

* Name
* Bio
* Social media links
* Profile picture
* Colors and theme

## Technologies

* TypeScript
* CSS3
* JavaScript
